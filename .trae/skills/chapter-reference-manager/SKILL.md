# Chapter Reference Manager

这个 Skill 用于在当前章节中插入一个简洁的提示，告知读者某个高深或复杂的知识点将在后续的哪个具体章节中详细学习。

## 使用场景
- 当文档中提到了一些复杂的概念，需要安抚读者并提供明确的学习路径时。
- 当用户明确指示“某个知识点会在哪一章学到”并要求补充到文档时。

## 规范格式

```markdown
> *💡 **注**：
> - **知识点**：“[具体知识点]” —— **对应章节**：[章节号] [章节完整名称]
> - **掌握内容**：[简要描述该章节将如何解决当前提到的知识点/疑惑]*
```

## 执行逻辑
1. **结构化展示**：使用引用块（`>`）配合列表形式。
2. **三位一体信息**：必须同时包含“**知识点名称**”、“**章节号+标题**”以及“**掌握内容**”，确保读者清晰知道什么知识点在哪里学、学到什么程度。
3. **内容提炼**：在“掌握内容”中，简要说明该章节将如何深入讲解（如：实战案例、底层原理、心智模型切换等）。
4. **位置选择**：通常插入在相关知识点描述的下方，或该小节的末尾。
5. **格式统一**：配合灯泡图标（`💡`）和加粗的“注”字。
